# Open WebUI

**Open WebUI 是一款[高度可扩展](https://docs.openwebui.com/features/plugin/)、功能丰富、操作便捷的自托管 AI 平台，完全支持离线运行。** 它不仅支持 **Ollama** 和 **OpenAI 兼容接口**等多种大语言模型运行环境，还内置了用于 RAG 的推理引擎，是一套**功能强大的 AI 部署解决方案**。

## Docker 快速上手指南 🐳

**如果您的电脑上已安装 Ollama**，只需运行以下命令：

```bash
docker run -d -p 3000:8080 --add-host=host.docker.internal:host-gateway -v open-webui:/app/backend/data --name open-webui --restart always ghcr.io/open-webui/open-webui:main
```

**需要启用 Nvidia GPU 支持**？请使用这个命令：

```bash
docker run -d -p 3000:8080 --gpus all --add-host=host.docker.internal:host-gateway -v open-webui:/app/backend/data --name open-webui --restart always ghcr.io/open-webui/open-webui:cuda
```

### Ollama 一体化安装包

我们提供了一个打包了 Open WebUI 和 Ollama 的一体化容器镜像，让您通过一条命令就能完成所有设置。根据您的硬件配置，选择合适的安装命令：

- **GPU 版本**：
如果您的设备配备了 GPU，可以使用以下命令充分利用硬件性能：

```bash
docker run -d -p 3000:8080 --gpus=all -v ollama:/root/.ollama -v open-webui:/app/backend/data --name open-webui --restart always ghcr.io/open-webui/open-webui:ollama
  ```

- **CPU 版本**：
如果您的设备没有 GPU，请使用这个命令：

```bash
docker run -d -p 3000:8080 -v ollama:/root/.ollama -v open-webui:/app/backend/data --name open-webui --restart always ghcr.io/open-webui/open-webui:ollama
  ```

这两个命令都会自动完成 Open WebUI 和 Ollama 的安装配置，让您可以立即开始使用所有功能。

安装完成后，打开浏览器访问 [http://localhost:3000](http://localhost:3000) 就能开始使用了。开始探索吧！😄

### 体验开发版本 🌙

:::warning
开发版（`:dev`）包含最新的实验性功能和更新，可能存在不稳定因素或未完成的功能，请谨慎使用。
:::

如果您想抢先体验最新功能，且能接受偶尔的不稳定，可以尝试使用 `:dev` 版本：

```bash
docker run -d -p 3000:8080 -v open-webui:/app/backend/data --name open-webui --restart always ghcr.io/open-webui/open-webui:dev
```

### 版本更新指南

以下是更新 Open WebUI 容器的简便方法：

#### 手动更新
借助 [Watchtower](https://containrrr.dev/watchtower) 工具手动更新容器：
```bash
docker run --rm -v /var/run/docker.sock:/var/run/docker.sock containrrr/watchtower --run-once open-webui
```

#### 自动更新
设置每 5 分钟自动检查并更新容器：
```bash
docker run -d --name watchtower --restart unless-stopped -v /var/run/docker.sock:/var/run/docker.sock containrrr/watchtower --interval 300 open-webui
```

🔧 **小贴士**：如果您使用了不同的容器名称，记得将命令中的 `open-webui` 替换为您的容器名称。

## 手动安装教程

Open WebUI 提供两种安装方式：使用 `uv` 运行时管理器或传统的 Python `pip` 包管理器。尽管两种方法都可行，但我们**强烈推荐使用 `uv`**，因为它能更好地管理环境依赖，大大减少潜在的冲突问题。

### 使用 `uv` 安装（推荐方案）

`uv` 运行时管理器为 Open WebUI 等应用提供了完善的 Python 环境管理。按照以下步骤开始安装：

#### 1. 安装 `uv`

选择适合您操作系统的安装命令：

- **macOS/Linux 系统**：
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
  ```

- **Windows 系统**：
```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
  ```

#### 2. 启动 Open WebUI

安装好 `uv` 后，启动 Open WebUI 就很简单了。使用下面的命令，注意设置 `DATA_DIR` 环境变量以保存数据。以下是不同平台的示例：

- **macOS/Linux 系统**：
```bash
DATA_DIR=~/.open-webui uvx --python 3.11 open-webui@latest serve
  ```

- **Windows 系统**：
```powershell
$env:DATA_DIR="C:\open-webui\data"; uvx --python 3.11 open-webui@latest serve
  ```

### 使用 `pip` 安装

如果您选择使用 Python 的 `pip` 包管理器安装 Open WebUI，**我们建议您配合使用 `uv` 或 `conda` 等 Python 环境管理工具**，这样可以更好地管理依赖关系，避免环境冲突。

目前，我们的开发环境是基于 Python 3.11。Python 3.12 虽然基本可用，但还未经过全面测试。至于 Python 3.13，由于完全未经测试，**使用时需要您自担风险**。

1. **安装步骤**：

在终端中执行：
```bash
pip install open-webui
   ```

2. **启动服务**：

完成安装后，执行以下命令启动服务：
```bash
open-webui serve
   ```

### 升级到新版本

需要升级到最新版本？只需一条命令：

```bash
pip install --upgrade open-webui
```

这种安装方式会自动处理所有必要的依赖项，让您能快速部署 Open WebUI。安装完成后，访问 [http://localhost:8080](http://localhost:8080) 即可开始使用。祝您使用愉快！😄
