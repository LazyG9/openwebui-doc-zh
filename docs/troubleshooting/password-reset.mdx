# 重置管理员密码 🗝️

如果你忘记了管理员密码，别担心！以下是针对 Docker 🐳 部署和本地安装的 Open WebUI 的分步重置管理员密码指南。

## Docker 部署方式 🐳

按照以下步骤重置使用 Docker 部署的 Open WebUI 的管理员密码。

### 步骤 1：生成新的密码哈希 🔐

首先，你需要为新密码创建一个 bcrypt 哈希。在本地机器上运行以下命令，将 `your-new-password` 替换为你想使用的密码：

```bash
htpasswd -bnBC 10 "" your-new-password | tr -d ':\n'
```

**注意：** 输出将包含带有特殊字符的 bcrypt 哈希值，需要小心处理。哈希值中的任何 `$` 字符都需要三重转义（替换为 `\\\`）才能在下一步中正确使用。

### 步骤 2：在 Docker 中更新密码 🔄

接下来，你将在 Docker 部署中更新密码。将下面命令中的 `HASH` 替换为步骤 1 中生成的 bcrypt 哈希值，确保对任何 `$` 字符进行三重转义。同时，将 `admin@example.com` 替换为与你的管理员账户关联的电子邮件地址。

**重要：** 以下命令可能不适用于所有情况。如果不起作用，请尝试下面的替代命令。

```bash
docker run --rm -v open-webui:/data alpine/socat EXEC:"bash -c 'apk add sqlite && echo UPDATE auth SET password='\''HASH'\'' WHERE email='\''admin@example.com'\''; | sqlite3 /data/webui.db'", STDIO
```

## 本地安装方式 💻

如果你有 Open WebUI 的本地安装，以下是如何直接在系统上重置管理员密码的方法。

### 步骤 1：生成新的密码哈希 🔐

与 Docker 方法一样，首先使用以下命令为新密码生成 bcrypt 哈希值。记得将 `your-new-password` 替换为你的新密码：

```bash
htpasswd -bnBC 10 "" your-new-password | tr -d ':\n'
```

### 步骤 2：在本地更新密码 🔄

现在，导航到本地机器上的 `open-webui` 目录。通过将 `HASH` 替换为步骤 1 中的 bcrypt 哈希值，将 `admin@example.com` 替换为你的管理员账户电子邮件，然后执行：

```bash
sqlite3 backend/data/webui.db "UPDATE auth SET password='HASH' WHERE email='admin@example.com';"
```


#### Docker 替代方法

_如果上述方法有问题。_ 我在 `alpine/socat` 中链接 `bash` 命令时遇到问题，_因为 `bash` 不存在。_

1. **运行连接到 open-webui 卷的 `alpine` linux。**

    ```bash
    docker run -it --rm -v open-webui:/path/to/data alpine
    ```
    _`/path/to/data` 取决于__你的__卷设置。_

    1. 安装 `apache2-utils` 和 `sqlite`：

        ```sh
        apk add apache2-utils sqlite
        ```
    1. 生成 `bcrypt` 哈希：

        ```sh
        htpasswd -bnBC 10 "" your-new-password | tr -d ':'
        ```

    1. 更新密码：

        ```sh
        sqlite3 /path/to/data/webui.db
        ```

        ```sql
        UPDATE auth SET password='HASH' WHERE email='admin@example.com';
        -- 退出 sqlite：[Ctrl + d]
        ```
