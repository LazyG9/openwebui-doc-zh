:::info **关于用户角色和隐私的重要说明：**

- **管理员创建：** 在 Open WebUI 上创建的第一个账户将获得**管理员特权**，可以控制用户管理和系统设置。
- **用户注册：** 后续注册的用户初始状态为**待审核**状态，需要等待管理员审核通过后才能正常访问系统。
- **隐私和数据安全：** **所有数据**，包括登录信息，都**完全存储**在您的本地设备上。Open WebUI 确保**严格的数据保密性**，不会发送任何外部请求，全面保障您的隐私和安全。
- **所有模型默认均为私有。** 只有通过群组共享或手动设置为公开的模型才能被其他人访问。若模型被分配给特定群组，则仅该群组成员可见。若模型被设置为公开，则该实例中的所有用户都可以访问。

:::

请根据您的需求选择以下安装方式：

- **Docker：** **官方推荐的安装方式，适合绝大多数用户**
- **Python：** 适合资源受限的环境或希望手动配置的用户
- **Kubernetes：** 适合需要大规模部署和容器编排的企业级用户

## 下一步

安装完成后，您可以通过以下地址访问：

- [http://localhost:3000](http://localhost:3000) 访问 OpenWebUI
- 如果您使用 Python 部署，请访问 [http://localhost:8080/](http://localhost:8080/)

现在您已经可以开始使用 OpenWebUI 了！
