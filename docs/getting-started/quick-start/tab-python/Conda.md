# Conda 环境安装指南

1. **创建 Conda 环境：**
   
   使用以下命令创建一个 Python 3.11 的新环境：
   ```bash
   conda create -n open-webui python=3.11
   ```

2. **激活 Conda 环境：**
   
   切换到新创建的环境：
   ```bash
   conda activate open-webui
   ```

3. **安装 Open WebUI：**
   
   使用 pip 安装最新版本：
   ```bash
   pip install open-webui
   ```

4. **启动 Open WebUI 服务：**
   
   运行以下命令启动服务：
   ```bash
   open-webui serve
   ```
