## DuckDuckGo API 配置

### 配置说明

使用 DuckDuckGo API 无需任何额外配置！它是 Open WebUI 的内置搜索引擎，开箱即用。

:::note
请注意：DuckDuckGo 可能会对频繁的搜索请求进行速率限制，以防止滥用。
:::
